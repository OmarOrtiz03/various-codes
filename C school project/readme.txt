Manual de uso de los programas

1. CC-Cadena 
esta parte contara el numero de semejanzas totales del texto que este dentro de "buscado.txt" en 
el texto de "entrada.txt" dando un archivo llamado "salida.txt" donde por cada linea 
maracara el numero de semejanzas totales del texto buscado.

nota: debe tener un texto de nombre "atributo.txt" donde dentro debe estar escrito "CC" para que funcione.

2.contadorWC
Esta parte contara cada una de las cadenas, lineas y caracteres que se encuentren dentro de un archivo
llamado "entrada.txt" y generara un archivo llamado "salida.txt" donde se dira exactamente cuantas
cadenas, lineas y caracteres se encuentran en el texto de la entrada.

nota: debe tener un texto de nombre "atributo.txt" donde dentro debe estar escrito "-WC" para que funcione.

3.contadorWNUM
Esta parte contara cada uno de los enteros que se encuentren dentro de un archivo llamado "entrada.txt" y 
generara un archivo llamado "salida.txt" donde se dira exactamente cuantos enteros se encuentran en el
texto de la entrada.

nota: debe tener un texto de nombre "atributo.txt" donde dentro debe estar escrito "-WNUM" para que funcione.

4.letraviejaletranueva
Esta parte reemplazara todas las ocurrencias de la letra indicada en el archivo "arch1.txt" por la letra 
indicada en el archivo "arch2.txt" dentro del texto del archivo "cadenas.txt" donde dara un archivo "salida.txt"
que estara mandando la cadena con el intercambio de letras.

nota: debe tener un texto de nombre "atributo.txt" donde dentro debe estar escrito "-S" para que funcione.

5.Mayus y Minus
Esta parte lo que hara es depende lo que diga un texto llamado "mayus.txt" donde "mM" comvertira las letras 
mayusculas a minusculas y "Mm" comvertira las letras minusculas a mayusculas de un archivo llamado "cadena.txt"
dara la comvercion en un archivo llamado "respuesta.txt".

6.NumerarLineas
Esta parte lo que hara es que convertira el archivo "entrada.txt" de forma que enumarara cada linea del texto
en orden dando la modificacion en un archivo llamado "salida.txt".

nota: debe tener un texto de nombre "atributo.txt" donde dentro debe estar escrito "-nL" para que funcione. 

7.Prefijo
Esta parte contara el numero de semejanzas totales del texto que este dentro de "buscado.txt" en 
el texto de "entrada.txt" dando un archivo llamado "salida.txt" donde por cada linea 
maracara el numero de semejanzas totales donde el texto buscado sea un prefijo.

nota: debe tener un texto de nombre "atributo.txt" donde dentro debe estar escrito "CP" para que funcione.

8.saltodelinea
Esta parte lo que hara es dar un salto de linea cuando en un archivo llamado "entrada.txt" ponga una "d"
y numero delante ejemplo: d1,d2,d3,d4.......

nota: debe tener un texto de nombre "atributo.txt" donde dentro debe estar escrito "IL" para que funcione.

9.Sufijo
Esta parte contara el numero de semejanzas totales del texto que este dentro de "buscado.txt" en 
el texto de "entrada.txt" dando un archivo llamado "salida.txt" donde por cada linea 
maracara el numero de semejanzas totales donde el texto buscado sea un sufijo.

nota: debe tener un texto de nombre "atributo.txt" donde dentro debe estar escrito "CS" para que funcione.

10.tabulacion
Esta parte lo que hara es dar una tabulacion cuando en un archivo llamado "entrada.txt" ponga una "d"
y numero delante ejemplo: d1,d2,d3,d4.......

nota: debe tener un texto de nombre "atributo.txt" donde dentro debe estar escrito "IC" para que funcione.
